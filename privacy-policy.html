1. Who We Are

This account migration service ("Service") is self-hosted software provided by [Your Name / Organization] ("Operator", "we", "us"). Contact: [your-email].

The underlying migration tool is open-source software (u-haul). The Operator is the data controller for data processed through this hosted instance.

2. What Data We Collect

To perform a migration we temporarily collect and process:

  • Email address – used to send migration notifications and the PLC verification token.
  • Source account handle – your current Bluesky/ATProto handle (e.g. alice.bsky.social).
  • Source account password – stored encrypted (AES-256-GCM) and automatically deleted within 48 hours, or immediately after a successful migration.
  • Target PDS host & new handle – the destination Personal Data Server you are migrating to.
  • Invite code (if required by target PDS) – stored encrypted and deleted after account creation.
  • PLC token – the one-time verification token emailed to you; stored encrypted and expires within 1 hour.
  • Migration token – an unguessable identifier (e.g. EURO-ABC12345) generated to let you track migration progress. No personal data is embedded in this token.
  • Technical log data – server logs may contain IP addresses and timestamps for operational purposes.

We do not collect: browsing history, analytics beyond those disclosed below, or any data unrelated to the migration.

3. How We Use Your Data

  • Execute the ATProto account migration on your behalf (the sole purpose).
  • Send transactional emails (migration start, completion, PLC token, backup ready).
  • Diagnose and recover from migration errors.

We do not use your data for marketing, profiling, or any purpose beyond operating the migration.

4. Data Retention & Deletion

  • Passwords & PLC tokens: Deleted immediately after use or within 48 h / 1 h respectively. Automatic cleanup jobs run every 6 hours.
  • Migration record: The non-sensitive migration record (status, timestamps, handle) may be retained for operational troubleshooting. Contact us to request deletion.
  • Email address: Retained only for the duration of the migration. Contact us to request deletion.
  • Backup bundles (if requested): Available for download for 24 hours, then automatically deleted from our server.
  • Server logs: Retained for up to [X] days for operational security, then purged.

5. Data Sharing

We do not sell, rent, or share your personal data with third parties, except:

  • Target PDS: Your account credentials are passed to the destination Personal Data Server you specify, which operates under its own privacy policy.
  • Source PDS (Bluesky / bsky.social): The migration tool reads your repository data from your current PDS.
  • Email provider: Transactional emails are sent via the SMTP provider configured by the Operator.
  • Legal requirements: We may disclose data if required by applicable law.

6. Security

Sensitive credentials are encrypted at rest using AES-256-GCM (Lockbox). Connections to this service use TLS. Rate limiting is applied to all endpoints. Despite these measures, no system is perfectly secure — use a strong, unique password.
